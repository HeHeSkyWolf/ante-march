# 赛博史记：安特进行曲

语言: [EN](./README.md) / [CN](./README_CN.md)

## 项目初衷

天狼维基将为您呈现游戏《安特进行曲》，安特进行曲是由**上海AWT工作室**开发，**7k7k小游戏**是其中一家发行商。
在这个维基中，我收集了我能从互联网上找到的关于游戏中的Boss、车辆、NPC和怪物的索引链接。希望您喜欢这些内容。

我发起这个项目的目的是为了存档并纪念我童年时代的一款网页游戏。这款游戏的服务器已经正式关闭，游戏内容也即将湮没在浩瀚的互联网信息海洋中。
这个项目旨在**存档并纪念**已于2015年正式关闭服务器的网页游戏《安特进行曲》。

## 如何启动？

1. 下载代码（通过下载压缩文件）。

2. 将压缩文件解压到您想要的目录。

3. 安装VSCode。
    - 这是官方网站链接: https://code.visualstudio.com/

4. 前往扩展程序，搜索`ritwickdey.LiveServer`，安装"Live Server"。
    - 这是扩展程序的ID: `ritwickdey.LiveServer`。
    - 请确保您安装的是正确的扩展程序！

5. 安装完成后，在VSCode窗口中，选择“文件”>“打开文件夹”，选择您在步骤2中解压的文件夹。

6. 在VSCode窗口中，打开文件“安特进行曲.html”。

7. 滚动到 VS Code 窗口底部，点击“Go Live”。

8. 天狼维基会运行在本地服务器上。
